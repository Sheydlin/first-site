Главный заголовок

Поискова система ситает его главным на странице, поэтому лучше использовать его только один раз.

Также заголовок h1-h6 занимает всю строку.

Работа с текстом:

Тег p — это абзац.
Тег br — новая строка
Тег b — жирный шрифт
Тег em — курсив

Тег a — ссылка. Атрибут может переводить по ссылке на другой сайт, или к определенному блоку на странице нашего сайта. Например: Мой ролик по игре ML:BB
Если нужно чтобы ссылка открывалась в новой вкладке, используем атрибут target. Например: Вэйл ML:BB
Для перехода к определенному блоку, тэгу на сайте, необхожимо присвоить ему уникальный id, затем в атрибуте тэга a указываем что это будет якорь, а не ссылка на другой сайт. Например; Вернуться в начало


Маркированный список

Для создания маркированного списка используется тег ul, а его элементы заключают в тег li. Также в этом списке можно добавлять ссылки. Например:

  • Первый элемент списка
  • Второй элемент списка
  • Яндекс Ссылка на главную странцу Яндекса
hr — тег является болчным и проводит горизонтальную черту в том блоке, в котором находится. Тег не закрывается.

Таблица в HTML

Для создания таблицы использется тег table, у него также есть шапка - thead,тело - tbody, и подвал - tfoot.
tr - строка, th- жирный текст по центру ячейки, td - обычный текст.
Атрибут colspan в теге td объединяет указанное кол-во ячеек. Атрибут border устанавливает рамку нашей таблице.

Используя тег img можно добавить изабражение на страницу сайта. Атрибут src отвечает за путь к изображению. Если файл находится на вашем компьютере, то достаточно указать относительный путь - он же имя файла. Если же файл находиться на другом сайте, то и путь нужно указывать абсолютный.
Ширина изображения регулируется атрибутом width, а высота атртбутом height.


Поле ввода добавляется с помощью тега input. Атрибут type - тип поля ввода (text, number, month).
